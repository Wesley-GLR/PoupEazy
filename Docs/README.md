## Estrutura da Documentação (Framework Diátaxis)

Esta documentação segue o framework [Diátaxis](https://diataxis.fr/), que organiza a documentação técnica em quatro categorias complementares:

```
Docs/
├── Tutoriais/        → Aprendizado orientado (learning-oriented)
├── How-tos/          → Guias práticos orientados a tarefas (task-oriented)
├── Referências/      → Informação técnica de consulta (information-oriented)
└── Explanações/      → Entendimento e contexto (understanding-oriented)
```
